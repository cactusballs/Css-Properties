Common Css Properties

and what they do

Name Syntax What do? Example
Color color: blue; this will set the foreground color of the selected element, usually the text. h1 { color: blue;}
Background Color background - color: red; this property sets the background color of a block of code, on this website it's set to #DDC192 which is the dark beige color you can see on the entire page, but in this table cell it's set to #E5771E. h1{ background-color: red;}
Font Family font-family: "Futura"; this property selects the type of font you would like to use for each element. You can also set back up fonts so your website will render on any machine. I've used Futura because it was a greta font before Supreme ever got involved but you can pick! h1 {font-family: "Futura", sans-serif; }
Text Align text-align: center; this property defines the horizontal alignment of text. You can use
left,
center or
right.
 h1 {
text-align: left;
}
Font Size font-size: 20px; this property specifies the size of the font. h1 { font-size: 90px; }
Font Weight font-weight: bold; this property specifies the thickness of the font. You can use a keyword e.g bold. A numeric number e.g 800 (equivalent to bold) or a relative value e.g bolder, this will make the element bolder than that of it's parent element. h1 { font-weight: 800px;}
Font Style font-style: italic; this property changes the style of the font. You can use normal, italic or oblique. h1 {font-style: oblique; }
Font Decoration font-decoration: underline; this property changes the text to use underline , line-through, overline etc. h1 { font-decoration: overline; }
Line Height line-height: 2.8; this sets the spacing between lines of text. two ways; 1.5 this would set the line height to 1.5 x the font size of the element or 150% would set the line height to 150% of the font size of the element. h1 {line-height: 1.4;}
